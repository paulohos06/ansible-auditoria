Esse projeto foi desenvolvido para obtenção do certificado de conclusão do IGTI/XP Educação - MBA em Segurança Cibernética. A ferramenta B.E - SecureIT, criada para a fintech fictítica B.E - Banco Emergente, é utilizada para hardening e auditoria de endpoints (estações de trabalho ou servidores) Linux. Ela é composta pelas ferramentas Audit, Lynis, OpenSCAP e orquestrada pelo Ansible como IaC. Para provisionamento do ambiente, sugere-se a utilização do Vagrant e do Oracle VirtualBox.
